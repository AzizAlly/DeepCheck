## Overview
DeepCheck is a deepfake detection system that uses machine learning to identify manipulated videos. It combines a Xception CNN with Bidirectional GRU layers to analyze both spatial and temporal inconsistencies in videos.

## Features
- 🎥 Video upload and analysis
- 🔍 Real-time deepfake detection
- 📊 Confidence scores for predictions
- 👤 User authentication system
- 📜 Prediction history tracking
- 🚀 FastAPI backend with React frontend

## Technology Stack
- **Backend**: FastAPI, TensorFlow, MongoDB
- **Frontend**: React, Tailwind CSS
- **ML Model**: Xception + Bi-GRU
- **Database**: MongoDB Atlas
